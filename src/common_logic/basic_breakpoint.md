# 普通断点

* 加断点
  * 常见2种方式
    * 通过函数名
      * 举例
        * C语言函数
          * `objc_alloc_init`
          * `UIApplicationMain`
        * iOS的OjbC函数
          * `-[NSString stringByAppendingString:]`
          * `-[AAUISignInController _performAuthenticationForAccount:serviceType:inViewController:completion:]`
    * 通过地址
      * 举例
        * `0x0000000100d5bbb0`
        * `0x1830c6a80`
